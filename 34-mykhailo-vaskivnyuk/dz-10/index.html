Функція index = removeValueOne(array, value, offset)

Шукає найближче value в array і видаляє в разі знаходження. Пошук здійснюється справа наліво з index = offset. При offset < 0 index = array.length - offset.

Функція повертає index видаленого елементу

Функція removeValueAll(array, value)

Шукає всі value в array і видаляє в разі знаходження. Пошук здійснюється справа наліво. Після видалення чергового знайденого елемента - пошук продовжується з index = removed_index - 1

Функція removeValues(array, ...values)

Шукає кожне value of values в array і видаляє всі знайдені елементи.

Функція unique_array = unique(array)

Повертає новий result_array, який містить лише унікальні значення з array

Функція diff_array1_2 = difference(array1, array2)

Повертає новий result_array, який містить значення з array1 і не містить жодного із значень з array2.
