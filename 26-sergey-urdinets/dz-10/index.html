TASK 1. Функция removeElement(array, item) для удаления элемента item из массива array. 

	function removeElement(array, value) {
		for (let i = array.length - 1; i >= 0; i--) {
			if (~array.indexOf(value)) {
				array.splice(array.indexOf(value), 1);
			}
		}
	}

TASK 3. Функция unique(array) возвращает новый массив, не содержащий дубликатов. 

		function unique(array){
			return array.filter((item, index) => array.indexOf(item) === index)
		}

TASK 4. Функция difference(array1, array2) находит разницу между массивами, т.е. возвращать новый массив, содержащий значения, которые содержались в array1, но не содержались в array2. 

		function difference(array1, array2) {
			return array1.filter(item => ! array2.includes(item, 0));
		}
